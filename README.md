# Proyecto de Juegos de Plataformas en Unity

¡Bienvenido al repositorio del proyecto de juegos de plataformas! Este proyecto es una prueba que demuestra la creación de un juego de plataformas utilizando el motor Unity. A continuación, encontrarás información relevante sobre el proyecto.

## Descripción

Este es un juego de plataformas 2D que permite a los jugadores explorar niveles, recoger objetos y superar obstáculos. El objetivo principal es llegar al final de cada nivel mientras se evitan enemigos y trampas.

## Características

- **Niveles diversos**: Incluye múltiples niveles con diferentes escenarios y desafíos.
- **Mecánicas de juego**: Saltos, movimientos y recogida de objetos.
- **Enemigos**: Diferentes tipos de enemigos con comportamientos variados.
- **Puntuación**: Sistema de puntuación basado en objetos recogidos y enemigos derrotados.

## Tecnologías Utilizadas

- **Unity 2026**: Motor principal del juego.
- **C#**: Lenguaje de programación utilizado para la lógica del juego.
